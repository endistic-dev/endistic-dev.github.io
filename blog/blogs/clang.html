Hey everyone!
This is my first blog, about me learning C.

C is a very... interesting language, to say the least.
Normally, programming languages give you a bunch of built-in
functions or data types. However, in C, you don't really get those
all too often. You mostly have to do it your self.

This also includes, you guessed it, memory management!
You have to dictate how much memory some variables can take up,
and stuff like pointers and memory addresses. Crazy stuff, I know.

In this blog, I will be documenting my experiences with C, and
linking some executable files alongside source code if you guys
want to try out some of the projects.

This is my first time blogging, so bare with me if this
is gonna be a bad blog.

Thanks for reading and have a good day!
